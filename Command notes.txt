<ul> to create a list of items in no particular order.

Individual list items must be added to the unordered list using the <li> tag. The <li> or list item tag is used to describe an item in a list.

example: 
<ul>
  <li>Limes</li>
  <li>Tortillas</li>
  <li>Chicken</li>
</ul>

 (<ol>) are like unordered lists, except that each list item is numbered.
example:
<ol>
  <li>Preheat the oven to 350 degrees.</li>
  <li>Mix whole wheat flour, baking soda, and salt.</li>
  <li>Cream the butter, sugar in separate bowl.</li>
  <li>Add eggs and vanilla extract to bowl.</li>
</ol>

	href: used to specify the destination URL.

	target:
		_blank: open the link in a new tab

	href: Can be used to link other html files. Mostly used to make the files more neat.
		For example: <a href="./contact.html">Contact</a>
